linear-gradient 线性渐变

background: linear-gradient(direction, color-stop1, color-stop2, ...); /*方向*/
background: linear-gradient(angle, color-stop1, color-stop2, ...); /*角度*/

为了创建一个线性渐变，你必须至少定义两种颜色结点。同时，你也可以设置一个起点和一个方向（或一个角度）。

以下示例为绿色到黄色渐变。

兼容写法，以左上到右下方向为例：

#grad {
    background: -webkit-linear-gradient(left top, green , yellow); /* Safari 5.1 - 6.0 */
    background: -o-linear-gradient(bottom right, green, yellow);   /* Opera 11.1 - 12.0 */
    background: -moz-linear-gradient(bottom right, green, yellow); /* Firefox 3.6 - 15 */
    background: linear-gradient(to bottom right, green , yellow);  /* 标准的语法 */
}

使用角度angle

direction 为预定义的角度，使用angle可以设置更灵活的角度。

#grad {
    background: -webkit-linear-gradient(180deg, green, yellow); /* Safari 5.1 - 6.0 */
    background: -o-linear-gradient(180deg, green, yellow);      /* Opera 11.1 - 12.0 */
    background: -moz-linear-gradient(180deg, green, yellow);    /* Firefox 3.6 - 15 */
    background: linear-gradient(180deg, green, yellow);         /* 标准的语法 */
}

使用多个颜色结点

三个颜色结点的渐变，还可以有更多个结点。

↓
background: linear-gradient(green, yellow, red);

使用透明度（Transparency）

背景绿色，透明度0到1渐变。为了添加透明度，使用 rgba() 函数来定义颜色结点。

↓
background: linear-gradient(rgba(0,128,0,0), rgba(0,128,0,1));

重复的线性渐变 repeating-linear-gradient

repeating-linear-gradient() 函数用于重复线性渐变。颜色后面的百分比或长度是该颜色的起始位置。

↓
background: repeating-linear-gradient(green, yellow 10%, red 20%);
↓
background: repeating-linear-gradient(green, yellow 25px, red 50px);

radial-gradient 径向渐变

background: radial-gradient(center, shape size, start-color, ..., last-color);
background: radial-gradient(green, yellow)
background: radial-gradient(green, yellow, red)
background: radial-gradient(green 5%, yellow 20%, red 60%)

    

    

        

        

    


    
设置形状

    
shape 参数定义了形状。它可以是值 circle 或 ellipse。其中，circle 表示圆形，ellipse 表示椭圆形。默认值是 ellipse。

    

        

        
background: radial-gradient(circle, green, yellow, red);

    

    

        

        
background: radial-gradient(ellipse, green, yellow, red);

    


    
不同尺寸大小关键字的使用

    
size 参数定义了渐变的大小。它可以是四个值：closest-side(近边)、farthest-side(远边)、closest-corner(近角)、farthest-corner(远角)，下面是四个不同的值的效果：

    

        

        

        

        

        
background: -webkit-radial-gradient(50% 50%, closest-side, green, yellow, red);

    

    

        

        

        

        

        
background: -webkit-radial-gradient(70% 60%, closest-side, green, yellow, red);

    


    
重复的径向渐变

    

        

        
background: repeating-radial-gradient(green, yellow 10%, red 15%);
