搭建项目

安装工具

第一步：安装 node 和 npm。

第二步：执行下面的命令安装 vue-cli（VUE的脚手架工具）

npm install -g vue-cli

用 vue-cli 构建一个项目

进入项目所在文件夹，执行下面的命令：

vue init webpack demo-project
执行过程中会出现以下问题： 
? Project name (demo-project)
? Project description (A Vue.js project)
? Author (xx <xx@xx.com>)

项目名、项目描述、作者，直接三个回车。

? Vue build (Use arrow keys)
❯ Runtime + Compiler: recommended for most users
  Runtime-only: about 6KB lighter min+gzip, but templates (or any Vue-specific HTML) are ONLY allowed in .vue files -
render functions are required elsewhere

是否需要安装编译器，回车

? Use ESLint to lint your code? (Y/n)

是否需要使用 ESLint 来检查你的代码，需要，回车。

? Pick an ESLint preset (Use arrow keys)
❯ Standard (https://github.com/feross/standard)
  Airbnb (https://github.com/airbnb/javascript)
  none (configure it yourself)

需要使用哪种风格来检查代码，回车选择Standard

? Setup unit tests with Karma + Mocha? (Y/n)
? Setup e2e tests with Nightwatch? (Y/n)

这两个问题是关于测试功能的，不需要，输入n，回车。

等待构建完成。

PS：一开始使用 PowerCmd 执行，出现? Project name (demo-project) 问题后卡住了，换成 git-bash.exe，顺利执行。

项目初始化

cd demo-project
npm install
npm run dev

其中 npm run dev 是测试运行项目，执行后可以通过 http://localhost:8080/ 访问示例页面，项目初始化完成。

修改ESLint规则，使用4个空格缩进

修改项目根目录下的 .eslintrc.js 文件，在 rules 中添加：

'indent': ['error', 4]
